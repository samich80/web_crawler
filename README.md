# web_crawler
web parser

Require:
Python 3.7. or above
Scrapy

Installing Scrapy:
pip3 install scrapy

Execution:

scrapy crawl bravebird (or scrapy crawl bravebird --loglevel="ERROR" - for less output) in project root directory

Result in items.jl
